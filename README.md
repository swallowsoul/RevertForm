RevertForm
---

#### Reclaim your humanity!

Features
 - In any situation where **Two Forms** or **Visage** is usable, pressing your "Toggle Sheath" (default Z) key will return you to human/visage form.

Macros/Addons
 - Can be called from a macro using: /click RevertFormButton LeftButton 1
 - Sets "RevertForm" as a boolean global.
